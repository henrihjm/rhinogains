# Icon Generation Guide

To generate PWA icons for Rhino Gains, you can use one of these methods:

## Method 1: Online PWA Asset Generator (Recommended)

1. Visit https://www.pwabuilder.com/imageGenerator
2. Upload a base icon (192x192 or larger)
3. Download the generated icon set
4. Place the following files in the `public/` directory:
   - icon-192x192.png
   - icon-512x512.png
   - icon-180x180.png (for iOS)
   - icon-152x152.png (for iOS)
   - icon-120x120.png (for iOS)

## Method 2: Manual Creation
